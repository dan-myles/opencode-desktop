## Updating Opencode SDK

1. Copy over new Opencode binaries
2. Run script to generate SDK `packages/sdk/js/script/generate`
3. Copy over new SDK
4. Add new endpoints to TRPC router

> Note: TRPC router API closely follows generated SDK
